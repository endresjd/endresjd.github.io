New website (using Publish)

The web site I have to provide information required by my mobile apps is very simple, but I did not like dealing with HTML or web site generators online. Too difficult for me given what little I have to offer on a web site. When I found John Sundell's Publish tool, I decided to play around with that. It let me focus on what little content I have, notably privacy policies, in a way that is more familiar. Markdown files. So this web site is now generated from a Swift package and updated as needed with my hosting provider.

Publish

Publish is a static site generator built specifically for Swift developers -- his words. My needs are simple and my updates rare, this lets me put things together using a collection of Markdown files that include hints in them to help Publish generate the website.
